# Twitter Clone 🐦

A Twitter/X clone built with React + Vite to understand 
what happens behind every click on a social media app.

## What I'm Building
- Tweet feed with multiple tweet cards
- Like button with real-time count update
- User profiles
- Post a tweet feature

## Tech Stack
- **Frontend:** React + Vite
- **Styling:** CSS
- **Backend (coming soon):** Node.js + Express
- **Database (coming soon):** PostgreSQL
